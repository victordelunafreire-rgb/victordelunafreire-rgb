## 👨🏽‍💻 Sobre mim

Após 15 anos liderando operações e equipes, migrei para o desenvolvimento web para construir, na prática, as soluções que sempre busquei como gestor. Minha missão não é apenas escrever código, mas traduzir regras de negócios complexas em aplicações que funcionem de ponta a ponta. Trago a maturidade de quem sabe resolver problemas sob pressão, unida à capacidade técnica de entregar interfaces responsivas e APIs bem estruturadas.

- 🚀 Foco atual no desenvolvimento Full Stack com o ecossistema JavaScript.
- 💡 **Projetos em destaque:** Sistema de Triagem Médica e Cardápio Digital Interativo.
- 🌍 **Idiomas:** Inglês (Fluente), Espanhol (Intermediário), Francês (Básico).
- 📫 **Como me encontrar:** [LinkedIn] http://linkedin.com/in/victordelunafreire ou e-mail.
